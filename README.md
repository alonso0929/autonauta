# AUTONAUTA Automation Framework

## Technologies

- Python
- Cucumber
- Selenium
- BeautifulSoup
- FastAPI

## Run the project

### API Test:
```
npm run apitest
```

### Web Test:
```
npm run webtest
```

### Server Integration Test:
```
npm run integrationtest
```

## Reports

### API Reports:
```
npm run report-apitest
```

### Web Reports:
```
npm run report-webtest
```
